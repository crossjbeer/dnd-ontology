A small Ontology for Dungeons and Dragons (DnD) built with OwlReady2 and RDFLib.

Our corpus is based on abstract, handwritten homebrew DnD materials.
ChatGPT was used to produce a YAML-formatted "Lore" document for ingestion into the ontology.

The ontology is minimal, focusing on Locations, Characters, Factions, and Items, while using Quests to link them in material ways. 

I purposefully built the ontology to demonstrate the use of several ObjectProperties, class hierarchies, and features of the Owlready2 package.
